# ShopSmart Grocery Web App

A full-stack grocery shopping web application built using:

- Python
- Flask
- MySQL
- HTML
- CSS
- JavaScript

## Features

- User Registration & Login
- Admin Login
- Product Management
- Cart Management
- Order History
- Inventory Management
- Responsive UI

## Installation

1. Clone the repository

```bash
git clone https://github.com/Amulya-456/ShopSmart-Grocery-Web-App.git
```

2. Install dependencies

```bash
pip install flask mysql-connector-python
```

3. Import `shopsmart.sql` into MySQL

4. Run the application

```bash
python app.py
```

5. Open browser
http://127.0.0.1:5000
